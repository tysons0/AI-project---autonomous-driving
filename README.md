#AI Autonomous Driving – Webots

This project extends the Webots sample autonomous driving world by adding:
- Custom obstacles (boxes and pedestrians)
- Camera and LiDAR-based perception
- Rule-based autonomous driving logic
- An ethical decision-making scenario



## How to Run
1. Open Webots
2. Open `worlds/ethical_autonomous_driving.wbt`
3. Press Play

## Key Features
- Camera-based object detection
- LiDAR obstacle avoidance
- Ethical prioritization of human safety
